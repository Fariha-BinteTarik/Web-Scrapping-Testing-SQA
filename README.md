# Web-Scrapping-Testing-SQA
# Book Scraper - Testing & Data Validation Documentation

## Project Overview
This project focuses on testing a Selenium-based web scraping system that extracts book data from an online bookstore.

The system collects:
- Book Title
- Price
- Availability

## Objective
To validate the accuracy, reliability, and performance of a web scraping system under different conditions.

## Features Tested
- Page navigation (multiple pages)
- Data extraction (title, price, availability)
- CSV file generation
- Command-line arguments handling
- Error handling when pages are unavailable

## Technologies Used
- Python
- Selenium WebDriver
- CSV Module
- Argparse

- Depends on website structure (HTML changes break scraper)

- Slower performance for large page counts
